OSMFILTER.EXE PARAMETER-FILE
-------------------------------
format used by customised EXTRACT_*.bat:
filter_<poi-name>.txt

filter_<poi-name>.txt contains the osm-tags (key-value) combinations to filter out and create your poi-extracts (csv,kml,osm)

Note: 
-use Notepad++ to view and edit filter_*.txt
-each argument/parameter in filter_*.txt should be separated by LINESPACE
-to have an idea how osmfilter works, see: http://wiki.openstreetmap.org/wiki/Osmfilter
-you can actually use osmfilter to extract almost anything from osm file (eg. administrative boundaries, routes, etc.)
